## 무료 문제

> 무료 문제 파일들은 **백준허브**를 통해 생성되었습니다.

0, 1, 2, 3, 4, 5 하위 파일은 **프로그래머스** 문제 난이도에 따라 분류한 파일입니다.   

난이도 파일의 하위 파일명은 **[번호].[문제명]** 입니다.   

문제 파일은 **문제 설명**을 담은 **README.md**과 **코드**를 담은 **[문제명].cpp**로 구성됩니다.   

> 일부 문제의 확장자는 **.c**, **.sql**로 되어 있습니다.

<hr/>

## 유료 문제

> 유료 문제는 문제 설명을 공개할 수 없어, **백준허브**를 사용하지 않고 직적 업로드하여 형식이 다릅니다.

하위 파일명은 **coding_[파트 번호]** 입니다.
특정 파트에 속한 파일명은 **test[파트번호]-[문제번호].cpp** 입니다.
